Bestiary Ancient Dossier UI Asset Pack

Contiene 70 PNGs transparentes para generar hojas de bestiario / dossier arcano en una app.

Categorías:
- Esquinas ornamentales
- Bordes de página
- Divisores horizontales
- Sellos rojos
- Símbolos arcanos
- Marcos de panel
- Marcadores decorativos

Uso sugerido:
Coloca los PNGs como overlays sobre un fondo de pergamino dentro de React Native / Expo.
Todos los archivos están pensados para conservar transparencia real en formato PNG RGBA.
